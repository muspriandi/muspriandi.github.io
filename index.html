Sqids Encoder / Decoder

Encode numbers into short unique IDs and back, powered by Sqids. Each section uses its own alphabet & minimum length. Everything runs in your browser.

🤖 Automation API (URL query)

Append query params to this page's URL to get a JSON result — no clicking. Handy for automated tests (drive it with a headless browser and read window.__sqidsResult).

Parameters

  • type = catalogcategory | item | variant
  • action = enc (numbers → ID) | dec (ID → numbers)
  • value = numbers (comma/space separated) for enc, or an ID for dec

Examples

?type=item&action=enc&value=12 ?type=catalogcategory&action=enc&value=42

Response is JSON: { "ok": true, "type": "item", "action": "enc", "result": "…" }. On failure: { "ok": false, "error": "…" }.
